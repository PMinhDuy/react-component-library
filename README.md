# React Component Library

> A reusable, accessible, and well-documented UI component library built with ReactJS and TypeScript.
> Inspired by real-world production work on a Singapore national digital health platform serving 100k+ MAU.

## 🚀 Features
- Accessible form components (WCAG compliant)
- Custom Ant Design wrappers with built-in validation
- Fully typed with TypeScript
- Tested with React Testing Library & Vitest

## 🛠️ Tech Stack
ReactJS · TypeScript · Ant Design · Tailwind CSS · Vitest · React Testing Library · Vite

## 📦 Getting Started
```bash
npm install
npm run dev
npm run test
```

## 📁 Project Structure
```
src/
  components/
    Form/
    Button/
    DataGrid/
    Modal/
  hooks/
  utils/
  types/
```

## 💡 Design Principles
- Composition over inheritance
- SOLID principles
- Accessibility-first (WCAG 2.1 AA)
- Zero breaking changes policy
